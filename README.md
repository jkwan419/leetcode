## How to Use This Repo

1. **Study by topic** — Work through one category at a time to build pattern recognition.
2. **Attempt first, then compare** — Try solving each problem on LeetCode before looking at the solution here.
3. **Review complexity** — Each solution notes its time and space complexity so you can evaluate trade-offs.
4. **Spaced repetition** — Revisit problems you found difficult after a few days to reinforce the patterns.
